# Install AnonAI for Notepad++

## Quick Install

1. Close Notepad++.
2. Create this folder if it does not exist:

```text
C:\Program Files\Notepad++\plugins\AnonAi
```

3. Copy `AnonAI.dll` into that folder.
4. Start Notepad++.
5. Open `Plugins > AnonAI > Settings...`.

## Portable Notepad++

Copy `AnonAI.dll` to:

```text
<portable-notepad++>\plugins\AnonAi\AnonAI.dll
```

## Shortcuts

Default shortcuts:

- `Ctrl+Alt+Shift+K`: Settings
- `Ctrl+Alt+Shift+A`: Instant answer
- `Ctrl+Alt+Shift+P`: Ask with extra prompt
- `Ctrl+Alt+Shift+H`: Secret typeout
- `Ctrl+Alt+Shift+X`: Stop

You can change them in Notepad++ via `Settings > Shortcut Mapper > Plugin commands`.
