# AnonAI 0.1.0

Release date: 2026-07-08

## Included

- Native Notepad++ plugin DLL.
- OpenAI Responses API provider.
- Claude Anthropic Messages provider.
- DeepSeek OpenAI-compatible provider.
- Custom OpenAI-compatible provider.
- DPAPI-protected API key storage.
- Instant answer command.
- Prompt window command.
- Secret typeout mode with cover character.
- Secret typeout auto-wrap based on visible editor width.
- Configurable persistent background chat history.
- Clear background chat history command.
- About dialog with creator credits.
